# User Journey Map
**Project:** MCE Platform
**Date:** 2025-12-28
**Purpose:** Visualize the steps users take to achieve their goals within the MCE Platform, identifying key touchpoints and system responses.

## Persona 1: Sarah (Sales Representative)
**Goal:** Respond to internal leads immediately and convert them.

| Stage | Activity | User Action | System Reaction | Touchpoint |
| :--- | :--- | :--- | :--- | :--- |
| **Discovery** | New Lead Arrival | Sarah is working on other tasks but keeps the MCE Dashboard open. | **System** ingests a new lead via Webhook. **System** auto-sends a "Welcome" WhatsApp msg. **Dashboard** updates Live Feed instantly. | Dashboard / WhatsApp |
| **Notification** | Alert & Review | Sarah sees the "Live Feed" flash with a new name. She clicks the lead card. | **System** opens the `ChatWindow` and loads the profile + history (including the auto-reply). | Lead Card |
| **Engagement** | Manual Follow-up | Sarah reads the context and types: "Hi, I see you're interested in X. Free for a call?" | **System** sends the message via `whatsapp-web.js`. **System** updates message status to one tick -> two ticks. | Chat Input |
| **Response** | Lead Replies | Sarah waits. The lead replies: "Yes, call me at 2pm." | **System** receives inbound msg. **System** updates UI with new bubble. **System** triggers "Stop Automation" (Kill Switch) to prevent bot interference. | Chat Window |
| **Resolution** | Conversion | Sarah calls the lead and marks status as "Qualified". | **System** updates `Leads` table status. **Analytics** increment "Conversation Rate". | Status Dropdown |

## Persona 2: Mike (Marketing Manager)
**Goal:** Re-engage a list of 500 cold leads using a blast campaign.

| Stage | Activity | User Action | System Reaction | Touchpoint |
| :--- | :--- | :--- | :--- | :--- |
| **Preparation** | Import Data | Mike uploads `cold_leads_nov.csv`. | **System** parses 500 rows. **System** skips 20 duplicates. **System** adds 480 new leads to DB. | Bulk Import UI |
| **Setup** | Campaign Creation | Mike navigates to "Campaigns", clicks "New". Enters name: "Nov Revival". Adds Step 1: "Hi {name}, long time..." | **System** generates Campaign ID. **System** saves steps to DB (Draft status). | Campaign Builder |
| **Launch** | Start Execution | Mike clicks "Start Campaign". | **System** filters eligible leads (non-stopped). **System** creates 480 Audience entries. **System** sets status to "Running". | Campaign Dashboard |
| **Monitoring** | Tracking Progress | Mike watches the Analytics tab. | **System** background worker processes queue. **System** updates "Sent" counter in real-time. | Analytics Page |
| **Outcome** | Review Results | Next day, Mike checks "Kill Switch" stats to see how many people replied. | **System** shows "15 Leads Stopped Automation" (Replied). Mike is happy with the 3% response rate. | Analytics/Reports |

## Persona 3: "The Bot" (System Automation)
**Goal:** Ensure zero lead leakage and hygiene.

| Stage | Trigger | Action | Outcome |
| :--- | :--- | :--- | :--- |
| **Ingestion** | Lead row added to Google Sheet. | **Poll Service** detects new row. Creates Lead. | Database populated. Workflow Triggered. |
| **Inactivity** | 24 Hours since last message. | **Campaign Worker** checks next step. Sends follow-up "Did you see this?". | Lead nudged. |
